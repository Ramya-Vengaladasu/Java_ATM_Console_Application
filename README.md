# ATM Console Application (Java)

## 📌 Project Overview

The ATM Console Application is a simple Java-based project that simulates the basic functionalities of an Automated Teller Machine (ATM) through a command-line interface.

This project was built using Java fundamentals such as variables, conditional statements, loops, methods, switch cases, and user input handling with the Scanner class.

---

## 🚀 Features

* Check Account Balance
* Deposit Money
* Withdraw Money
* Input Validation
* Menu-Driven Interface
* Repeat Transactions
* Exit Option

---

## 🛠️ Technologies Used

* Java
* Scanner Class
* Conditional Statements
* Loops
* Methods
* Switch Case
